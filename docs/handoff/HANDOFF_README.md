# Handoff Directory — Resumen y Maturidad

Este directorio contiene artefactos que describen contratos de ownership, runbooks operativos y reglas de entrega entre sprints.

Estado de madurez de documentos (citar HANDOFF_MATURITY en cada archivo):
- packet_ownership_tracing.md — MATURE
- CI_GATING_RULES.md — MATURE
- RUNBOOK_PACKET_LEAK_DIAGNOSIS.md — DRAFT

Uso
- Mantener estos documentos actualizados en cada sprint. Los cambios que impacten contratos deben pasar por aprobación de Arquitecto y aparecer en ACTIVE_RISKS.md.
